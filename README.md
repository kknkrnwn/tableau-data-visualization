A collection of interactive Tableau dashboards showcasing data visualization, business intelligence, and data-driven insights using public datasets from Kabupaten Cirebon.

---

## 📖 Overview

This repository contains Tableau dashboards developed to explore and visualize data from various sectors, including education, demographics, health services, livestock production, public administration, and emergency services.

The goal of these dashboards is to transform raw data into meaningful insights through interactive visualizations and effective data storytelling.

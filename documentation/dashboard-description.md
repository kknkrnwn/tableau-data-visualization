# Dashboard Summary

This repository contains a collection of Tableau dashboards developed using public datasets from Kabupaten Cirebon. The dashboards aim to provide data-driven insights through interactive visualizations that support analysis, monitoring, and decision-making.

---

## 1. Student Distribution by Gender and School Status

### Dashboard

**Jumlah Siswa Berdasarkan Jenis Kelamin - Status Sekolah di Kabupaten Cirebon**

### Objective

Analyze the distribution of students based on gender and school status within Kabupaten Cirebon.

### Key Insights

* Comparison between male and female student populations.
* Distribution across public and private schools.
* Identification of gender balance patterns.
* Support for educational planning and policy evaluation.
